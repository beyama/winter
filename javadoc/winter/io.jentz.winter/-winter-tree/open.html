fun open(vararg path: Any, identifier: Any? = null, block: ComponentBuilderBlock? = null): Graph

Create a dependency graph by (sub-)component path and register it. Opened components will be children of each other in left to right order.

Parameters

path - The path of the (sub-)component to initialize.

identifier - An optional identifier to store the sub dependency graph under.

block - An optional ComponentBuilderBlock that's passed to the (sub-)component init method.

Exceptions

WinterException - When application component is not set or path can not be resolved.

IllegalArgumentException - When path is empty (root) but identifier is given.

Return
The newly created and registered graph.
