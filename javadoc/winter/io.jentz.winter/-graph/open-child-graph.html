openChildGraph

fun openChildGraph(subcomponentQualifier: Any, identifier: Any? = null, block: ComponentBuilderBlock? = null): Graph

Opens and returns a child graph by using the subcomponent with subcomponentQualifier and registers it under the subcomponentQualifier or when given under identifier.

The resulting graph gets automatically disposed when this graph gets disposed.

Parameters

subcomponentQualifier - The qualifier of the subcomponent.

identifier - An optional identifier to register the graph with.

block - An optional builder block to register provider on the subcomponent.
